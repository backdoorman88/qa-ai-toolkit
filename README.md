# QA AI Toolkit

A Python-based API testing tool with LLM integration for automated response analysis.

## What it does

- Sends HTTP requests to REST API endpoints
- Validates response status codes and data fields
- Detects anomalies using AI analysis

## Who it's for

QA engineers who want to automate API testing and integrate AI into their workflow.

## How it works

HTTP Request → Response Validation → LLM Analysis → Report

## Tech stack

- Python 3.13
- requests
- pytest
- python-dotenv

## Run

```bash
python3 main.py
```
